# Neovim Flake

A Nix flake for managing a custom Neovim setup with [NVF](https://github.com/NotAShelf/nvf).

## Usage
Clone: `git clone https://github.com/SibaldH/nvim-flake`

Run: `nix run .` to launch Neovim with the config in flake.nix
